Euterpe Music Player é uma aplicação desktop desenvolvida em C# utilizando o framework .NET 8, voltada para a reprodução de arquivos de áudio com qualidade. O projeto foi concebido para fornecer uma experiência fluida de reprodução musical, com suporte a diversos formatos, incluindo FLAC, garantindo alta fidelidade sonora para os usuários. A aplicação possui uma interface gráfica moderna construída com WPF, permitindo navegar por álbuns, visualizar capas e informações de músicas de maneira intuitiva.
